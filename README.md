לימוד קריאת הפטרת שבת זכור בנוסח כורדיסטאן

לעילוי נשמת מו"ר אבי יעקב חנוכה בן זריפה ועילוי נשמת אימי צפורה סעדה בת יאזי
נלב"ע ביום י"א באדר התשפ"ה 11/03/2025 למניינם

הכנתי מיני אתר זה כדי ללמוד את הקריאה בימי האזכרה, השנה יוצא היראצייט י"א באדר של אבי יעקב חנוכה בן זריפה בדיוק בשבת זכור, אבא היה לוחם דגול מלוחמי היחידה 101 ומקימי גדוד 890 ולחם חזק חזק בעמלקים, מתאים שתיקרא לזכרו הפטרה זאת.

לימוד קריאת הפטרת שבת זכור בנוסח כורדיסטאן

לעילוי נשמת אימי צפורה סעדה בת יאזי ואבי יעקב חנוכה בן זריפה ועילוי נלב"ע ביום י"א באדר התשפ"ה 11/03/2025 למניינם

הכנתי מיני אתר זה כדי ללמוד את הקריאה בימי האזכרה, השנה התשפ"ו יוצא היראצייט י"א באדר של אבי יעקב חנוכה בן זריפה בדיוק בשבת זכור את אשר עשה לך עמלק, אבא היה לוחם דגול מלוחמי היחידה 101 ומקימי הצנחנים גדוד 890 ולחם חזק חזק בעמלקים, מתאים שתיקרא לזכרו הפטרה זאת.

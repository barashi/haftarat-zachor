# haftarat-zachor
לימוד קריאת הפטרת פרשת זכור בנוסח כורדיסטאן

לעילוי נשמת מו"ר אבי יעקב חנוכה בן זריפה ועילוי נשמת אימי צפורה סעדה בת יאזי

הכנתי מיני אתר זה כדי ללמוד את הקריאה בימי האזכרה
 השנה יוצא היראצייט י"א באדר של אבי יעקב חנוכה בן זריפה בדיוק בשבת זכור, אבא היה לוחם דגול מלוחמי היחידה 101 ומקימי גדוד 890 ולחם חזק חזק בעמלקים .
